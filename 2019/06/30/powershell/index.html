powershell 美化

Posted on 2019-06-30 | In 工具

让powershell 变得好看一点

字体

在中文环境下，默认 PowerShell 的新宋体很丑。然而由于默认 PowerShell 终端是一个非常底层的应用，其界面甚至没有利用 WinForm、WPF 等通用 UI 渲染框架来实现，而是直接调用底层 Windows API 来实现，因此其字体要求非常严格。

目前据我所知，唯一支持这一严格要求的字体（中文环境下）只有「Sarasa Gothic / 更纱黑体 / 更紗黑體 / 更紗ゴシック」（后面简称更纱黑体），可以从 更纱黑体的 GitHub页面下载。
